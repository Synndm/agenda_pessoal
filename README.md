# 📅 Agenda Pessoal com JavaScript

Este projeto consiste em uma aplicação web interativa para gerenciamento de tarefas, permitindo ao usuário adicionar, visualizar e remover atividades de forma dinâmica.

## 🚀 Demonstração
🔗 https://syn-agenda-pessoal.vercel.app/

## 📌 Sobre o projeto
O objetivo deste projeto foi desenvolver uma aplicação prática utilizando JavaScript puro, aplicando conceitos de manipulação do DOM, lógica de programação e armazenamento de dados no navegador.

A aplicação funciona como uma agenda pessoal, permitindo ao usuário organizar suas tarefas de forma simples e eficiente.

## 🛠 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 💡 Funcionalidades
- Adição de tarefas
- Listagem dinâmica de tarefas
- Remoção de tarefas
- Manipulação do DOM em tempo real
- Armazenamento de dados no navegador (localStorage)

## 🎯 Objetivo
Desenvolver habilidades em:
- Lógica de programação
- Estrutura de aplicações web
- Manipulação de listas e dados
- Persistência de dados com localStorage

## 📚 Aprendizados
Durante o desenvolvimento deste projeto, foram praticados conceitos como:
- CRUD (Create, Read, Delete)
- Manipulação do DOM
- Eventos em JavaScript
- Armazenamento local de dados
- Atualização dinâmica da interface
